Detailed Description

Visitor interface for all concrete AST node kinds.

This interface is the single dispatch surface for compiler passes that operate on AST nodes. Each pass implements these visit methods and receives a concrete node type through double-dispatch via ASTNode::accept().

Why explicit visit overloads?
Explicit overloads provide type-safe traversal and avoid runtime casts in each compiler phase.
