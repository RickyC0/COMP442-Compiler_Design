Detailed Description

LL(1) recursive-descent parser implementation.

This source maps grammar productions to parsing functions and builds AST nodes while recording derivation traces and recoverable syntax errors.

Why this implementation style?
Direct one-function-per-production code makes grammar debugging and assignment verification straightforward.
What comes next?
Outputs from this module (AST + syntax diagnostics) feed semantic analysis.
